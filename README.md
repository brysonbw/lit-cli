# lit-cli

A CLI tool to scaffold and manage Lit projects.

[![Crates.io](https://img.shields.io/crates/v/lit-cli?style=flat)](https://crates.io/crates/lit-cli)
[![Crates.io](https://img.shields.io/crates/d/lit-cli?style=flat)](https://crates.io/crates/lit-cli)
![CI](https://img.shields.io/github/actions/workflow/status/brysonbw/lit-cli/ci.yml?branch=main&style=flat&logo=github&label=CI)

## Install

```bash
cargo install lit-cli
```

## Usage

```bash
lit-cli <COMMAND>
```

### Commands

| Command | Description                                               |
| ------- | --------------------------------------------------------- |
| new     | Create a new project                                      |
| help    | Print this message or the help of the given subcommand(s) |

### Options

| Option        | Description   |
| ------------- | ------------- |
| -h, --help    | Print help    |
| -V, --version | Print version |
